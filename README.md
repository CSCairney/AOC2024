## Getting Started

### Prerequisites
Make sure you have the following tools installed on your system:

- **Node.js** (for TypeScript)
- **Java Development Kit (JDK)** (for Java)
- **Rust Compiler (rustc)** (for Rust)

---

### Running Solutions

#### TypeScript
1. Navigate to the specific day's TypeScript folder:
   ```bash
   cd day-1/typescript
   ```
2. Install dependencies (if applicable):
   ```bash
   npm install
   ```
3. Run the solution:
   ```bash
   npm start
   ```

#### Java
1. Navigate to the specific day's Java folder:
   ```bash
   cd day-1/java
   ```
2. Compile the program:
   ```bash
   javac Main.java
   ```
3. Run the solution:
   ```bash
   java Main
   ```

#### Rust
1. Navigate to the specific day's Rust folder:
   ```bash
   cd day-1/rust
   ```
2. Build and run the solution:
   ```bash
   cargo run
   ```
